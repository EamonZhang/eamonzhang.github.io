## 运行

- cmd command
- regedit 注册表
- calc 计算器
- notepad 笔记本
- mspaint 图画板
- winver window版本
- shutdown 60秒关机


## cmd

- ipconfig /all 查看 ip信息
- cd            切换目录
- D:            切换到D目录
- dir           当前目录下文件信息
- ipconfig  /flushdns 刷新本地dns缓存
- ipconfig  /displaydns 显示dns缓存
