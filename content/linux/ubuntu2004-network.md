## 配置

vim /etc/netplan/00-installer-config.yaml 
```
# This is the network config written by 'subiquity'
network:
  ethernets:
    enp2s0:
      addresses:
      - 192.168.6.111/24
      gateway4: 192.168.6.1
      nameservers: 
        addresses: [119.29.29.29]
  version: 2
```

## 生效
```
netplan apply
```
